# AS5600 Nema 17 Mounts

Mounting hardware for attaching the AS5600 rotary encoder onto the back of the NEMA 17 stepper motor
Includes a magnet centering jig which helps you properly center the magnet onto the rear shaft to glue it in place

## Mounting Options:
### 1. Custom PCB + Spacers
Parts required:
- Diametric magnet
- Super glue (or equivalent strong adhesive)
- 4x M3 Machine Screws
- Custom AS5600 breakout board
- 4x Spacers
